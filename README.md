(> BGCS Scenario Practice Questions Website)
This website is a simple and friendly version to a beginners guide to understanding some foundational cyber security basics. Learn and touch up on commands, knowledge, and test yourself with provided questions.
The prototype started as a simple questionare that users were able to answer, but the final version connected the extra elements including tabs, and a whole cybersecurity "gloassary."

(Functions):
Tab Learn (Glossary): A small glossary that focuses on Linux and Powershell commands, and discusses more information ergarding Networking and overall Cybersecurity.
Tab Practice (scenarios): Interactive questions that the user can test their knowledge and test commands in their OS's terminal.

(How To Run):
The project uses asynchronous Javascripts 'fetch()'s to retrieve information from the local JSON files (commands.json) + (CyberSecurityFoundation.json).
1. The user will need to download the zip file or clone this project to the User's local machine.
2. Open the project in Visual Studio Code, and ensure the user has a live server host extension.
3. If done correctly, the user should be able to access the '((.)) Go Live' at the bottom right corner of the user's studio code.
4. The user finally will right-click on the 'index.html' file, and press the '((.)) Go Live' button.
5. Have fun! The user should be able to have the proper user access of the website.
